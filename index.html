Foosball Detection

The foosball ball tracking object detection model is an artificial intelligence-based system developed specifically for monitoring and analyzing foosball games. This model performs object detection by tracking the movement of the ball on the foosball table and continuously updates the position of the ball. This is useful for better understanding foosball matches, evaluating players' strategies and providing data for potential performance improvements.

The main function of the model is to use the YOLOv7 (You Only Look Once) object detection algorithm to analyze the video stream and detect the pinball ball. This algorithm works very quickly and effectively to detect objects in the image and determine their positions. Using these object detection results, the model continuously tracks the motion of the pinball.

The success of the model is quite impressive, as tracking and detecting a fast-moving ball on a foosball table is quite a challenging task. The 80% success rate shows that the model works successfully despite this challenge. This provides a reliable tool for studying and analyzing pinball games.

The ability to track a foosball ball offers potential for a range of applications. For example, analyzing foosball matches can help players improve their strategy. It can also be used for educational purposes, helping beginners to understand the basics of the game of pinball. It can also be used for analyzing professional foosball tournaments and evaluating player performance.

This object detection model that tracks a foosball ball stands out as an example of how AI technology can be used in sports analysis and training. Such technologies can be used in a wider range of application areas and offer new perspectives in the world of sports.

Park Area Detection

In my project, I developed a program to detect parking lots using image processing techniques. This program works with important Python-based libraries such as numpy and pandas. In the first step, we take the images from the camera and convert them to black and white format. This is an important step to simplify and speed up the process.

When working on a black-and-white image, we use corner and edge detection algorithms to identify regions that remain in white. This step allows us to identify each parking lot slot in the image. We then determine whether the parking lot is empty or occupied by calculating the pixel count of these white regions.

The threshold is a critical component of this detection process. The threshold determines how many white pixels should be present. When the threshold is exceeded, we know that a vehicle is in the parking lot. As a result of this process, we detect empty and occupied parking lots.

Finally, we save these detection results in a SQL database file. This database file contains information about the status of each parking slot. Thus, users or administrators can monitor the status in the parking lot at any time and know about the free spaces.

This project provides a great convenience in parking lot management and supervision. Thanks to image processing techniques and database management, continuous monitoring and tracking of parking lots becomes possible. Advanced algorithms and artificial intelligence technologies can also be integrated to improve and further optimize this project. In this way, parking lot management can be made more effective and efficient.
